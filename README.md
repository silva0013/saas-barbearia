# SaaS Barbearia

This project is a web application for managing barbershop appointments, clients, and barbers. It includes a frontend built with React and Vite, and a backend using Node.js and Express.

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd saas-barbearia
   ```
2. Set up environment variables for the backend by creating a `.env` file from the `.env.example` in the `backend` folder.
3. For the frontend, create an `.env` file from the `.env.example` in the `frontend` folder.
4. Run Docker Compose:
   ```bash
   docker-compose up
   ```
5. Navigate to the frontend folder and start the frontend:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
6. Navigate to the backend folder and start the backend:
   ```bash
   cd backend
   npm install
   npm run start
   ```
